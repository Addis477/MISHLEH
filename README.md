# MISHLEH · House of Wisdom

A sacred journey of spiritual formation. Free. Open Source. For the Church.

## Features
- 📜 **The Upper Room** - 21-day guided journey with Solomon
- 🔨 **The Workshop** - Practical discipleship projects
- 📓 **The Scriptorium** - Private journal with cloud sync
- 🙏 **Prayer Wall** - Anonymous community prayer
- 🕯️ **Stillness Timer** - Guided meditation with Scripture
- 📱 **PWA** - Install on any device

## Setup
1. Clone this repo
2. Create a Firebase project (free)
3. Replace `firebaseConfig` in `index.html`
4. Deploy to GitHub Pages

## Build APK
Push to `main` branch. GitHub Actions automatically builds `.apk` file.
Download from Actions tab → Artifacts.
